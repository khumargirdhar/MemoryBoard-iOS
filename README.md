# MemoryBoard-iOS
MemoryBoard is your personal memory keeper, capturing life's moments with words, images, and tags—all wrapped in a beautifully intuitive interface. Dive into memories by date with the calendar view, seamlessly relive past experiences, and keep everything safe and synced with Apple's UserDefaults & FileManager. Effortlessly preserve your journey, one entry at a time. ✨
